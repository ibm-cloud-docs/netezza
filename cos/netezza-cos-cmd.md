# Command line interface enhancements
{: #netezza_cmd}

The `nzds` command provides information about dataslice capacity and usage:

- For **Block Storage**, it displays:
  - Total capacity
  - Current usage percentage
  
- For **Object Storage**, it displays:
  - Used size only
