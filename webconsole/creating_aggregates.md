# Creating aggregates
{: #create-aggregates}

You cannot create aggregates with the web console. Instead, use the [CREATE {OR REPLACE} AGGREGATE](https://www.ibm.com/docs/en/netezza?topic=npssr-create-replace-aggregate-2) command.
